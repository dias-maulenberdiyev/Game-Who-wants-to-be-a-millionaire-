# Millionaire Game
This is a simple Python quiz game based on "Who Wants to Be a Millionaire".

The player answers 6 questions.  
Correct answers increase the prize, wrong answers end the game.

Safe levels:
- £100 after question 2
- £1000 after question 4

Run the game:
python millionaire_game.py
